This is a demo of the wasm build of geopattern, a rust crate by me. geopattern is a rust implementation of the geo_pattern ruby library by Jason Long

This demo is built using wasm-pack, a tool to export JavaScript modules for rust crates built using wasm-bindgen.

The example is based on the SHA1 digest example at hellorust, except instead of a SHA1 digest, this shows the geopattern.
