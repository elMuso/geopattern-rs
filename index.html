This is a demo of the wasm example in geopattern, a rust crate by me. geopattern is a rust implementation of the geo_pattern ruby library by Jason Long

The example itself is built on rust's wasm32-unknown-unknown target. Please see the associated Makefile for build details.

The example is based on the SHA1 digest example at hellorust, except instead of a SHA1 digest, this shows the geopattern.
